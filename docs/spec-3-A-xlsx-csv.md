- ステータス: **確定**（2026-07-14、モックイメージ提示 → AskUserQuestion 3巡＋追加1巡で全論点合意）
- 関連: `docs/品質向上ロードマップ.md` 3-A／実装: `index.html`・`xlsx-export.js`・`main.js`・`preload.js`／テスト: `test/xlsxcsv.test.js`・`test/xlsxexport.test.js`

## 目的

日本の現場の点検表・チェック記録は Excel 文化であり、Word 出力だけでは提出フローに
乗らない。エディタから .xlsx / .csv をワンクリックで書き出せるようにする。
ゼロコスト方針のため exceljs（MIT・無料 OSS）を使用する。

## 確定事項（ユーザー決定）

| # | 論点 | 決定 |
|---|------|------|
| 1 | 対応範囲 | Excel・CSV とも **Electron 専用**。ブラウザ/単体HTMLではボタン自体を出さない（`window.xlsxAPI`/`window.csvAPI` の有無で出し分け）。ロードマップ当初案の「ブラウザは Blob ダウンロード」はこの決定で上書き |
| 2 | チェック列 | 出力時点の状態を **✓／空欄** で出す。一括リセット後に出力すれば未記入の点検様式になる |
| 3 | 画像 | **Excel に埋め込む**。各手順の直下に「画像行」を1行挿入し、その手順の全画像を横並び。行高は最大の1枚に合わせる。大きさは**トリミング後の原寸を保持**し、幅 960px（A4縦の印刷幅相当）を超える分のみ縦横比を保って縮小。画像の無い手順には画像行を挿入しない。CSV には含めない |
| 4 | 本文(body) | リッチテキスト本文を**プレーンテキスト化して「詳細」列**に入れる（装飾は除去、ブロック要素・`<br>` はセル内改行に） |
| 5 | ToDo 型 | **テンプレートと同じ列構成**（標準時間・メモ・詳細は空欄）。複数リストを並べても列が揃う |
| 6 | シート先頭/末尾 | 1行目タイトル（結合・太字）、2行目メタ「種別／進捗 x/y 完了（z%）／標準時間合計 n分／出力日」＋表紙有効時は文書番号・版数・作成者を追記、3行目列見出し「No／項目／チェック／標準時間(分)／メモ／詳細」（ウィンドウ枠固定）、セクションごとに結合・色付き見出し行、最下段に合計行 |
| 7 | 表紙 | **表紙が有効なテンプレートのみ**、シート1に「表紙」シート（タイトル・サブタイトル・作成者・日付・版数・文書番号の設定済み項目＋ロゴ画像＋改訂履歴表）。本体はシート2。無効・ToDo は1シート構成 |
| 8 | CSV | BOM 付き UTF-8・CRLF・RFC 4180 エスケープ。1行目に列見出し、以降**データ行のみ**（タイトル・メタ・合計行なし）。列は「セクション,No,項目,チェック,標準時間(分),メモ,詳細」 |
| 9 | 名前 | ファイル名・シート名とも**タイトル由来**。シート名は Excel 制約に合わせ禁止記号 `: \ / ? * [ ]` を `_` に置換し31文字で切り詰め（表紙シートは固定「表紙」） |

## 実装上の補足既定

- Excel の行高上限は 409pt（≒545px）のため、高さ 540px を超える画像も縦横比を保って縮小する。
- exceljs が扱えない画像形式（webp 等。通常は取り込み時に JPEG/PNG へ正規化済み）はスキップする。
- 画像の横並びは列グリッドの px 幅近似（`列幅×7+5`）による小数 col アンカー指定のため、横位置は数 px ずれうる（「横に並ぶ」ことが要件であり許容）。
- No はセクション横断の通し番号（画面・印刷ビューと同じ規則）。
- ファイル名の sanitize・保存ダイアログ・戻り値の形（`{saved}/{canceled}/{error}`）は既存の Word 出力（`docx:save`）と同型。

## アーキテクチャ

Word 出力と同じ三層構成。

1. **レンダラー（index.html）**: 純関数で出力仕様を組む。
   - `buildXlsxSheetData(checklist)` → `{ sheetName, cover, columns, rows }`（rows は kind 付き行配列。画像は dataURL＋px 寸法）
   - `buildCsvText(checklist)` → BOM 込みの完成 CSV 文字列
   - `htmlToPlainText(html)`／`excelSheetName(title)`／`capImageSize(w,h)` を補助に持つ
   - 画像寸法は `measureXlsxImages(rows)` が `new Image()` で取得して埋める
2. **IPC**: `xlsx:save`（data を渡し main 側で構築・保存）／`csv:save`（文字列を渡し保存のみ）。preload で `xlsxAPI`/`csvAPI` を公開。
3. **main.js ＋ xlsx-export.js**: `buildXlsxWorkbook(ExcelJS, data)`（新規モジュール。ExcelJS 注入でテスト可能）でワークブックを構築し、ネイティブ保存ダイアログで書き出す。
