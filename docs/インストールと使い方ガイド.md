# CheckListMaker 使い始めガイド（Windows）

ビルド済みのインストーラ（`.exe`）をダウンロードして、CheckListMaker を使い始めるまでの
手順です。**Node.js や開発ツールは不要**で、インストーラを実行するだけで使えます。

---

## ① ダウンロード

最新のインストーラは GitHub Actions の**ビルド成果物（Artifact）**として配布しています。
下記の「ダウンロードページ」を開き、一番新しい成功ビルドから取得してください。

**▶ ダウンロードページ（最新の成功ビルド一覧）:**
<https://github.com/kuwagata-137/CheckListMaker/actions/workflows/build-windows.yml?query=is%3Asuccess>

手順:

1. 上のリンクを開く（GitHub へのログインが必要です。あなたのリポジトリなので問題ありません）。
2. 一覧の**一番上にある最新の実行**（緑のチェック ✅）をクリック。
3. 実行ページの一番下までスクロールし、**「Artifacts」**の
   **`CheckListMaker-Windows-installer`** をクリック → zip がダウンロードされます。
4. zip を解凍すると、**`CheckListMaker Setup 1.0.0.exe`** が入っています。

> 💡 各ビルドの実行ページ上部「Summary」にも、その回のダウンロードリンクが自動表示されます。
>
> ⚠ Artifact は**約90日で期限切れ**になります。期限切れ・リンク切れのときは、上のページで
> 最新ビルドを開き直すか、ビルドを再実行（Re-run）してください（再実行すると新しい
> インストーラが作られます）。

## ② インストール

1. ダウンロードした **`CheckListMaker Setup 1.0.0.exe`** をダブルクリック。
2. **管理者権限は不要**です（現在のユーザー向けにインストールされます）。
3. 自動的にインストールされ、**デスクトップ**と**スタートメニュー**に
   「CheckListMaker」のショートカットが作られます。

導入先（参考）: `%LOCALAPPDATA%\Programs\CheckListMaker\`

### 初回起動時の警告（SmartScreen）について

このインストーラは**コード署名をしていない**ため、初回に Windows の SmartScreen が
「Windows によって PC が保護されました」と警告を出すことがあります。問題ありません。

- 「**詳細情報**」をクリック → 「**実行**」を押すと起動できます。

---

## ③ 使い方の基本

1. ショートカットから **CheckListMaker** を起動。
2. **「＋ 新規作成」**でリストを作成（手順書を作るなら**テンプレート型**が便利）。
3. 手順・項目を入力。見出し（フェーズ/セクション）でグループ分けもできます。
4. **録画**したいときは、右上の **⏺ 録画ボタン**を押す。
   - 以降、**クリックするたびに画面全体のスクリーンショットが自動保存**されます。
   - 画面右上に録画ガジェット（経過時間・枚数・プレビュー・停止）が出ます。
     ガジェット自体はスクショには写りません。
   - 停止は、ガジェットの「■ 録画停止」または ⏺ の再押下。
5. 撮った画像は各手順の **🖼 画像**から貼り付け、サムネイルをクリックすると
   **注釈エディタ**（矢印・四角・モザイク・トリミング・テキスト等）で書き込めます。
   - 編集画面は広く、**編集中だけの表示ズーム**（🔍 スライダー＋数値入力）で
     拡大して細かく描けます（拡大縮小は表示だけで、保存される画像の仕上がりは変わりません）。
   - 手順に貼った画像は**手順欄の幅いっぱい**に大きく表示されます。
6. 完成した手順書は、ブラウザの**印刷→PDF保存**や、**リンク共有**で配れます。

### スクリーンショットの保存先

録画で撮った画像は、アプリ実行ファイルと同じフォルダ直下の
**`00_スクリーンショット`** に保存されます（導入先がユーザー書き込み可のため、
管理者権限なしで保存できます）。

- ファイル名: `<リスト名>_yyyymmdd_<連番>.png`

## 困ったときは

- **ダウンロードリンクが切れている** → 上の「ダウンロードページ」で最新ビルドを開くか、
  ビルドを再実行してください。
- **録画ボタンが押せない（無効）** → ブラウザで `index.html` を直接開いた場合は録画が
  使えません。インストール版（このガイドの方法）を使ってください。
- **独自アイコンにしたい／コード署名したい** → 開発者に依頼してください
  （`build/icon.ico` の追加や署名証明書の設定で対応できます）。
