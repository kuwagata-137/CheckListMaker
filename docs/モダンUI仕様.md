# モダンUI リニューアル仕様（方向D + 影エフェクト）

CheckListMaker の見た目フルリニューアルにあたり、着手前に確定した仕様をここに記録する。
実装は `index.html` の `<style>`（CSS カスタムプロパティ＝デザイントークン）を中心に行い、
DOM 構造・JavaScript ロジックは変更しない。書き出し HTML／`.docx` は `<style>` をライブ取得
しているため、本改修は自動的に反映される。

## 確定した決定事項

- **方向**：D（プロ / ダッシュボード）。スレート系の落ち着いた配色、アクセントは青1色。
- **影エフェクト**：強さは「**やや強め**（strong）」。適用範囲は以下**すべて**。
  1. **ツールバー（`#topbar`）下**に影を入れ、本体より前面に見せる。
  2. **項目ホバー**時にリフト（背景＋影＋わずかな浮き上がり）。
  3. **セクションのカード化**（本アプリでは既にカード。影を強めて浮きを明確化）。
- **依存**：ゼロのまま維持（外部 CDN・Web フォント・アイコンライブラリを追加しない）。
- **範囲**：見た目（配色・タイポ・質感）のみ。機能・データ構造・操作フローは不変。

## デザイントークン

アクセントは `#2F6FEB`（ダークは `#4A8DFF`）の1色。進捗バー・完了マーク・ネイティブ
チェックボックス（`accent-color`）もこのアクセントに統一する。

### ライト
| トークン | 値 | 用途 |
|---|---|---|
| `--bg` | `#eef1f5` | ページ背景（カードが浮くトレイ） |
| `--surface` | `#ffffff` | カード面 |
| `--surface-2` | `#eff2f6` | 補助面・ホバー |
| `--text` | `#1c2430` | 本文（スレート系インク） |
| `--muted` | `#5f6b7a` | 補助テキスト |
| `--border` | `#dfe3e8` | 境界線 |
| `--border-strong` | `#c4cbd3` | 強めの境界（チェック枠など） |
| `--primary` / `--accent` | `#2f6feb` | アクセント（1色） |

### 影（多層ソフトシャドウ・やや強め）
| トークン | ライト | ダーク |
|---|---|---|
| `--shadow`（面の resting） | `0 2px 4px rgba(16,24,40,.06), 0 10px 24px rgba(16,24,40,.12)` | `0 2px 5px rgba(0,0,0,.5), 0 14px 30px rgba(0,0,0,.55)` |
| `--shadow-hover`（カードのリフト） | `0 6px 14px rgba(16,24,40,.10), 0 18px 38px rgba(16,24,40,.17)` | `0 8px 18px rgba(0,0,0,.5), 0 22px 44px rgba(0,0,0,.62)` |
| `--shadow-item`（項目ホバー） | `0 2px 8px rgba(16,24,40,.12)` | `0 3px 10px rgba(0,0,0,.55)` |

## コンポーネント別の適用

- **`#topbar`**：`box-shadow: var(--topbar-shadow)` を付与（sticky のまま前面感を出す）。
- **ホーム `.card`**：resting は `--shadow`、hover で `--shadow-hover` ＋ `translateY(-2px)`。
- **`.section`**：`--shadow` で浮きを強調（既にカード。ホバーでの移動はしない）。
- **`.item`（ToDo項目）**：hover で `--shadow-item` ＋ `translateY(-1px)`。`.item.step`（テンプレート
  手順）は従来どおりリフトさせない（`box-shadow:none; transform:none`）。
- **ネイティブチェックボックス**：`accent-color: var(--primary)`。
- **進捗バー**：トラックに内側の凹み（inset shadow）、バーにアクセントの淡い発色影。
- **`.btn.primary`**：アクセントの淡い影＋ hover で軽く暗く。
- **モーション**：`prefers-reduced-motion: reduce` でトランジション／リフトを無効化。
- **フォーカス**：ボタン等に `:focus-visible` のリング（アクセシビリティ）。

## 非対象（今回は触らない）

- 印刷ビュー・表紙（カバーページ）の配色：紙前提の固定色のため据え置き。
- リッチ本文エディタの表色（`#eef4fb` 等）：機能面の見え方に影響するため据え置き。
- DOM 構造・JS・データ形式・キーボード操作。
